# Auto Job Apply

Automated LinkedIn and Naukri job application bot built with Selenium WebDriver and TestNG.

## 📋 Overview

This project automates the job application process on two major job portals:
- **LinkedIn** - Automated job search and application
- **Naukri** - Automated job search and application

The automation framework uses Selenium WebDriver for browser automation and TestNG for test execution, allowing for parallel test execution and comprehensive reporting.

## 🛠️ Tech Stack

- **Language:** Java
- **Build Tool:** Maven
- **Testing Framework:** TestNG 7.4.0
- **Browser Automation:** Selenium 4.15.0
- **WebDriver Management:** WebDriverManager 5.5.3
- **Data Handling:**
    - Apache POI 5.2.4 (Excel files)
    - Gson 2.10.1 (JSON)
    - Jackson 2.15.2 (JSON/XML)
    - JSON Simple 1.1.1
- **Logging:** Log4j 1.2.17 & Log4j 2.20.0
- **Reporting:** Allure 2.24.0, ReportNG 1.1.4
- **REST API Testing:** Rest-Assured 4.3.0
- **OCR:** Tess4j 5.8.0
- **Utility:** Lombok 1.18.36, Apache Commons Text 1.10.0
